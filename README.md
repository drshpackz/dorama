# Дорама — плагин для Lampa

Плагин добавляет пункт **«Дорама»** в левое меню Lampa и открывает большую живую
ленту южнокорейского кино и сериалов из TMDB: все жанры (романтика, комедии,
драмы, экшен, исторические, фэнтези, мистика) плюс отдельные тематические
подборки — тёмные социальные триллеры (в духе «Паразитов», «Игры в кальмара»),
психологический хоррор и триллеры-головоломки. Сверху — **персональная лента
«Рекомендации для Вас»** по вашим лайкам с процентом совпадения, затем «Сейчас
смотрят», самое популярное и свежие новинки.

Лента **динамичная**: при каждом открытии ряды слегка обновляются (показывается
другая страница каталога), причём узор зависит от категории и ваших прошлых
оценок — так вы не видите одно и то же снова и снова.

> **Только корейский язык.** Все ряды отфильтрованы по `with_original_language=ko`,
> поэтому **японские тайтлы (например «Exit 8») не появятся** — это осознанный
> выбор каталога. Как включить японское/азиатское — см. «Как настроить выдачу».

## Установка

1. Откройте Lampa → **Настройки → Расширения** (Settings → Расширения).
2. Добавьте плагин по ссылке (raw URL):
   ```
   https://drshpackz.github.io/dorama/dorama.js
   ```
3. Перезапустите Lampa. В левом меню появится пункт **«Дорама»**.

## Что внутри

Один экран с рядами карточек (сериалы и фильмы), сверху вниз:

1. **Рекомендации для Вас** — персональные рекомендации по вашим лайкам/оценкам,
   с бейджем «Совпадение xx%».
2. Сейчас смотрят
3. Самые популярные: фильмы / Самые популярные: сериалы
4. Корейские новинки: сериалы / Корейские новинки: фильмы
5. Тёмные триллеры (исходные подборки): Корейские триллеры (сериалы), Корейское
   кино: триллеры, Социальные триллеры (неравенство), Выживание и антиутопия,
   Дом-ловушка (бетон / многоэтажка), Игры разума и саспенс, Лучшее: корейские
   триллеры.
6. Тематические: Психологический хоррор (сериалы / фильмы), Триллер-головоломка
   (сериалы), Триллер-головоломка: игра со зрителем (фильмы).
7. Жанровый каталог: Романтические дорамы, Корейские комедии, Korean drama:
   драмы, Боевики и экшен, Исторические фильмы (сагык), Фэнтези и мистика,
   Лучшее корейское кино, Лучшие корейские сериалы.

Карточки открывают обычную страницу деталей Lampa; у рядов-каталогов есть «ещё»
— полная сетка с бесконечной прокруткой (кроме ленты рекомендаций).

## Лайки и рекомендации

- Нажмите и удерживайте **OK** на карточке → выберите **«Нравится»** (нативное
  избранное Lampa).
- Лайкнутые корейские/азиатские тайтлы становятся основой ленты **«Рекомендации
  для Вас»**: плагин берёт TMDB-рекомендации по каждому лайку и оценивает
  кандидатов **строго по вкусу** — оставляет только азиатское и только с
  совпадением по жанрам с вашими лайками, а вес рейтинга/популярности понижен,
  чтобы наверх не лезла попса. Топ показывается с бейджем **«Совпадение xx%»**.
  Уже лайкнутые и просмотренные тайтлы исключаются, а «не понравившиеся» (реакции
  😴/💩 и их похожие) опускаются по всему каталогу.
- Пока лайков нет — в ряду показывается приглашение лайкнуть дорамы. Чем больше
  лайков, тем точнее рекомендации (обновляются при следующем открытии).

## Как настроить выдачу

Ряды лежат в `dorama.js`: `buildRows()` (исходные тёмные триллеры, не трогаются),
`popularRows()` («сейчас смотрят» + популярное), `buildExtraRows()` (тематические
+ жанровый каталог) и `buildDynamicRows()` (новинки на датах). Можно:

- **Расширить выдачу** — поднять/убрать `vote_count.gte` или убрать
  `with_keywords`.
- **Сменить тон** — поменять `with_genres` (помните: жанры **Триллер `53`** и
  **История `36`** существуют только для фильмов, не для сериалов `discover/tv`).
- **Включить японское/азиатское (в т.ч. «Exit 8»)** — в нужных рядах заменить
  `with_original_language=ko` на `with_original_language=ja|ko` (или
  `ko|ja|zh|th`). Рекомендации (`ASIAN_LANGS` в `dorama.js`) уже допускают
  `ja/zh/th`, так что лайкнутое японское и так попадёт в персональную ленту.
- **Настроить динамику** — `rowPage(...)` задаёт ротацию страниц; глубина задаётся
  полем `depth` у ряда (по умолчанию 3, у топ-рейтинга/новинок 2), `rotate:false`
  выключает ротацию для узких рядов. Счётчик открытий хранится в
  `Storage('dorama_open_seq')`.
- **Скорость загрузки** — `ROW_CONCURRENCY` (по умолчанию 4) — сколько рядов
  грузится параллельно; при ошибках 429 от прокси TMDB снизьте до 3.
- **Скрытие контента (BL / gay)** — все ряды каталога идут с
  `without_keywords=<BL_KEYWORDS>` (boys' love, lgbt, gay, homosexuality и т.п.),
  а лента рекомендаций дополнительно исключает такие тайтлы по id (`buildBLBlock`,
  т.к. `/recommendations` не принимает `without_keywords`). Чтобы изменить набор —
  правьте константу `BL_KEYWORDS` в `dorama.js` (id ключевых слов TMDB).
- **Настроить рекомендации** — берётся до 8 последних позитивных сигналов
  (`collectSignals().positives.slice(0, 8)`); веса в `scoreCandidate`
  (совпадение между лайками / жанр / язык / рейтинг) и формула `predictionPercent`
  определяют «процент совпадения». Жёсткие фильтры «только азиатское» и
  «обязательное совпадение жанров» — в цикле `finish()` внутри
  `loadRecommendations`.

## Совместимость

Чистый ES5, зависит только от глобального `Lampa` и `$` (jQuery из Lampa).
Никаких внешних запросов, кроме TMDB (через встроенный источник `tmdb`).
