# Дорама — плагин для Lampa

Плагин добавляет пункт **«Дорама»** в левое меню Lampa и открывает подборку
тёмных корейских социальных триллеров (в духе «Паразитов», «84 квадратных
метра», «Украденной личности», «Игры в кальмара») из TMDB. Сверху — **персональная
лента «Рекомендации для Вас»**, которая строится по вашим лайкам и показывает
процент совпадения; ниже — проверенные подборки TMDB Discover.

## Установка

1. Откройте Lampa → **Настройки → Расширения** (Settings → Расширения).
2. Добавьте плагин по ссылке (raw URL):
   ```
   https://drshpackz.github.io/dorama/dorama.js
   ```
3. Перезапустите Lampa. В левом меню появится пункт **«Дорама»**.

## Что внутри

Один экран с рядами карточек (сериалы и фильмы):

1. **Рекомендации для Вас** — персональные рекомендации по вашим лайкам, с
   бейджем «Совпадение xx%».
2. Корейские триллеры (сериалы)
3. Корейское кино: триллеры
4. Социальные триллеры (неравенство)
5. Выживание и антиутопия
6. Дом-ловушка (бетон / многоэтажка)
7. Игры разума и саспенс
8. Лучшее: корейские триллеры

Карточки открывают обычную страницу деталей Lampa; у рядов-каталогов есть «ещё»
— полная сетка с бесконечной прокруткой (кроме ленты рекомендаций).

## Лайки и рекомендации

- Нажмите и удерживайте **OK** на карточке → выберите **«Нравится»** (нативное
  избранное Lampa).
- Лайкнутые корейские/азиатские тайтлы становятся основой ленты **«Рекомендации
  для Вас»**: плагин берёт TMDB-рекомендации по каждому лайку, оценивает
  кандидатов по жанрам, языку, рейтингу и совпадению между лайками и показывает
  топ с бейджем **«Совпадение xx%»**. Уже лайкнутые и просмотренные тайтлы
  исключаются.
- Пока лайков нет — в ряду показывается приглашение лайкнуть дорамы. Чем больше
  лайков, тем точнее рекомендации (обновляются при следующем открытии).

## Как настроить выдачу

Запросы лежат в функции `buildRows()` в `dorama.js`. Можно:

- **Расширить выдачу** — поднять/убрать `vote_count.gte` или убрать
  `with_keywords`.
- **Сменить тон** — поменять `with_genres` (помните: жанр **Триллер `53`
  существует только для фильмов**, не для сериалов `discover/tv`).
- **Добавить регион** — скопировать ряд и поменять `with_original_language`
  (`ja` — Япония, `zh` — Китай, `th` — Таиланд).
- **Настроить рекомендации** — в `dorama.js`: `collectSeeds(liked, 8)` задаёт,
  сколько последних лайков берётся за основу; веса в `scoreCandidate`
  (жанр / язык / рейтинг / совпадение между лайками) и формула
  `predictionPercent` определяют итоговый «процент совпадения».

## Совместимость

Чистый ES5, зависит только от глобального `Lampa` и `$` (jQuery из Lampa).
Никаких внешних запросов, кроме TMDB (через встроенный источник `tmdb`).
